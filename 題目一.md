## 題目一：常微分方程與脈衝函數

### 題目
$$\frac{d^2x}{dt^2} - 3\frac{dx}{dt} + 2x = \delta(t), \quad \text{初始條件：} x(0) = 0, x'(0) = 0$$

### 詳細解答步驟

本題最適合使用**拉普拉斯轉換 (Laplace Transform)** 來求解。

**步驟 1：對微分方程兩邊取拉普拉斯轉換**
設 $\mathcal{L}\{x(t)\} = X(s)$。根據微分性質與脈衝函數的轉換公式 $\mathcal{L}\{\delta(t)\} = 1$：
$$\left[ s^2 X(s) - s x(0) - x'(0) \right] - 3 \left[ s X(s) - x(0) \right] + 2 X(s) = 1$$

**步驟 2：代入初始條件**
將 $x(0) = 0$ 和 $x'(0) = 0$ 代入上式：
$$s^2 X(s) - 3s X(s) + 2X(s) = 1$$
$$(s^2 - 3s + 2) X(s) = 1$$

解出 $X(s)$：
$$X(s) = \frac{1}{s^2 - 3s + 2} = \frac{1}{(s-1)(s-2)}$$

**步驟 3：部分分式拆解 (Partial Fraction Decomposition)**
將分式拆解為兩個簡單分式之和：
$$\frac{1}{(s-1)(s-2)} = \frac{A}{s-1} + \frac{B}{s-2}$$

利用留數法（Heaviside cover-up method）求係數：
* 求 $A$：令 $s = 1$，得到 $A = \frac{1}{1-2} = -1$
* 求 $B$：令 $s = 2$，得到 $B = \frac{1}{2-1} = 1$

因此：
$$X(s) = \frac{1}{s-2} - \frac{1}{s-1}$$

**步驟 4：取反拉普拉斯轉換 (Inverse Laplace Transform)**
根據反轉換公式 $\mathcal{L}^{-1}(\frac{1}{s-a}) = e^{at}$，可得單位脈衝響應解：
$$x(t) = e^{2t} - e^t \quad (t \ge 0)$$

### 最終答案
$$x(t) = e^{2t} - e^t$$
