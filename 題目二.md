## 題目二：傅立葉級數展開

### 題目
令 $f(x) = x^2$，其中 $x \in [-\pi, \pi]$ 且 $f(x + 2\pi) = f(x)$。求其傅立葉級數。

### 詳細解答步驟

在 $[-\pi, \pi]$ 區間內，週期為 $2\pi$ 的傅立葉級數公式為：
$$f(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} \left[ a_n \cos(nx) + b_n \sin(nx) \right]$$

**步驟 1：分析函數對稱性**
因為 $f(-x) = (-x)^2 = x^2 = f(x)$，所以 $f(x)$ 是一個**偶函數 (Even Function)**。
* 偶函數與正弦函數（奇函數）相乘後的積分為 0，因此所有正弦項係數：
  $$b_n = 0 \quad (n = 1, 2, 3, \dots)$$

我們只需要計算 $a_0$ 與 $a_n$。

**步驟 2：計算直流項係數 $a_0$**
$$a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} x^2 dx = \frac{2}{\pi} \int_{0}^{\pi} x^2 dx$$

$$a_0 = \frac{2}{\pi} \left[ \frac{x^3}{3} \right]_0^{\pi} = \frac{2}{\pi} \left( \frac{\pi^3}{3} \right) = \frac{2\pi^2}{3}$$

---

**步驟 3：計算餘弦項係數 $a_n$**
$$a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} x^2 \cos(nx) dx = \frac{2}{\pi} \int_{0}^{\pi} x^2 \cos(nx) dx$$

這裡我們使用**分部積分法（或連鎖積分表格法）**：

交叉相乘並考慮正負號可得：
$$\int_{0}^{\pi} x^2 \cos(nx) dx = \left[ x^2 \frac{\sin(nx)}{n} - 2x \left(-\frac{\cos(nx)}{n^2}\right) + 2 \left(-\frac{\sin(nx)}{n^3}\right) \right]_0^{\pi}$$

將上限 $\pi$ 與下限 $0$ 代入（注意 $\sin(n\pi) = 0$ 且 $\sin(0) = 0$）：
$$= \left( 0 + \frac{2\pi \cos(n\pi)}{n^2} - 0 \right) - (0 + 0 - 0)$$

$$= \frac{2\pi (-1)^n}{n^2} \quad (\text{因為 } \cos(n\pi) = (-1)^n)$$

將其代回 $a_n$ 的公式中：
$$a_n = \frac{2}{\pi} \cdot \frac{2\pi (-1)^n}{n^2} = \frac{4(-1)^n}{n^2}$$

---

**步驟 4：寫出最終級數展開式**
將 $a_0$ 與 $a_n$ 代入傅立葉級數公式，注意第一項是 $\frac{a_0}{2}$：
$$f(x) = \frac{\frac{2\pi^2}{3}}{2} + \sum_{n=1}^{\infty} \frac{4(-1)^n}{n^2} \cos(nx)$$

### 最終答案
$$f(x) = \frac{\pi^2}{3} + \sum_{n=1}^{\infty} \frac{4(-1)^n}{n^2} \cos(nx)$$
