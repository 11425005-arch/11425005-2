# 工程數學作業：拉普拉斯反轉換求解

## 題目描述

求解下列拉普拉斯反轉換方程：

$$\mathcal{L}^{-1} \left[ \frac{1 - e^{-3s}}{s(s + 4)} \right]$$

---

## 解題過程

### Step 1：拆解原式
利用拉普拉斯轉換的線性性質，將分子拆開為兩項：
$$\mathcal{L}^{-1} \left[ \frac{1 - e^{-3s}}{s(s + 4)} \right] = \mathcal{L}^{-1} \left[ \frac{1}{s(s + 4)} \right] - \mathcal{L}^{-1} \left[ \frac{e^{-3s}}{s(s + 4)} \right]$$

### Step 2：部分分式展開 (Partial Fraction Expansion)
我們需要先計算核心項 $\frac{1}{s(s + 4)}$ 的部分分式：
$$\frac{1}{s(s + 4)} = \frac{A}{s} + \frac{B}{s + 4}$$

利用蓋利得法（Cover-up method）求係數：
* 當 $s = 0$ 時，$A = \frac{1}{0 + 4} = \frac{1}{4}$
* 當 $s = -4$ 時，$B = \frac{1}{-4} = -\frac{1}{4}$

因此：
$$\frac{1}{s(s + 4)} = \frac{1}{4} \cdot \frac{1}{s} - \frac{1}{4} \cdot \frac{1}{s + 4}$$

### Step 3：求解第一項的反轉換
利用基本拉普拉斯反轉換公式 $\mathcal{L}^{-1}\left[\frac{1}{s}\right] = 1$ 以及 $\mathcal{L}^{-1}\left[\frac{1}{s - a}\right] = e^{at}$：
$$\mathcal{L}^{-1} \left[ \frac{1}{s(s + 4)} \right] = \frac{1}{4} - \frac{1}{4}e^{-4t}$$
我們令這個基本時間函數為 $g(t) = \frac{1}{4} (1 - e^{-4t})$。

### Step 4：求解第二項（利用平移定理）
根據第二平移定理（時間平移）：$\mathcal{L}^{-1}\left[e^{-as}F(s)\right] = f(t - a)u(t - a)$，其中 $u(t)$ 為單位階梯函數（Unit Step Function）。

本題中 $a = 3$，故將 $g(t)$ 中的 $t$ 全部替換為 $(t - 3)$，並乘上 $u(t - 3)$：
$$\mathcal{L}^{-1} \left[ \frac{e^{-3s}}{s(s + 4)} \right] = g(t - 3)u(t - 3) = \frac{1}{4} \left( 1 - e^{-4(t - 3)} \right) u(t - 3)$$

### 最終答案
將兩項合併，得到最終解為：
$$f(t) = \frac{1}{4}(1 - e^{-4t}) - \frac{1}{4}\left(1 - e^{-4(t - 3)}\right) u(t - 3)$$
